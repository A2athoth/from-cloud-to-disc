# Evolution Note: 2.0.0

## Repository state reviewed

At 1.9.0, the archive had gained ten major conceptual layers since 1.0.1. The risk was no longer absence, but navigational sprawl.

A second major version should not add another large subject. It should make the accumulated structure understandable.

## Changes made

- Updated version metadata to 2.0.0.
- Regenerated file inventory and checksums.

## Result

The archive is larger than before, but its layers are now explicitly mapped. This is a good stopping point for the second expansion wave.
