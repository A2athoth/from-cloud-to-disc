# Evolution Note: 0.4.0

## Repository state reviewed

At 0.3.0, the repository was easier to enter, but the compact disc existed mostly as an idea.

## Reconsideration

If the archive is about moving from cloud to disc, the disc should have its own textual presence.

The repository should imagine labels, sleeves, volume identifiers, and booklet-like notes while remaining plain text.

## Changes made

- Added case, sleeve, volume, and booklet files under `disc/`.
- Added a Korean note for the disc layer.
- Preserved the no-binary, no-image constraint.

## Result

The CD is no longer only a metaphor. It has a small textual body inside the archive.
