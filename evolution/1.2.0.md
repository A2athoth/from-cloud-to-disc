# Evolution Note: 1.2.0

## Repository state reviewed

At 1.1.0, the archive had a curation model, but its museum-like potential was still implicit.

## Reconsideration

The repository is an artifact, so some of its parts should be readable as exhibits.

## Changes made

- Added an exhibits layer.
- Described the repository, commit, disc, README, and checksum as display objects.
- Added a Korean guide for the exhibits layer.

## Result

The archive became easier to read as a small textual museum.
