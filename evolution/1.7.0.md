# Evolution Note: 1.7.0

## Repository state reviewed

At 1.6.0, the archive had strong human-readable layers, but structured summaries could help machines and quick readers.

## Reconsideration

Structured data should support the archive without replacing the human text.

## Changes made

- Added a data folder.
- Added JSON, CSV, and keyword views of the archive.
- Added fileset definitions for minimal, Korean, disc, and machine-readable views.

## Result

The archive became easier to inspect programmatically while keeping plain text as the primary medium.
