# Evolution Note: 0.3.0

## Repository state reviewed

At 0.2.0, the repository had a strong archive core but expected readers to find their own path through it.

## Reconsideration

The artifact should not assume that every reader arrives through the same entrance.

Some may arrive from GitHub. Some may arrive from a disc. Some may read Korean first. Some may only want the shortest explanation.

## Changes made

- Added bilingual reading paths.
- Added bilingual FAQ documents.
- Updated version metadata and checksums.

## Result

The repository became easier to enter without weakening its plain-text identity.
