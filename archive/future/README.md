# Future Reading

This page gathers the future-facing notes that were previously split across several short files. It is for readers who encounter the archive after its original context has faded.

## How to Read This if Context Is Lost

Do not begin by asking what application this repository runs.

It runs no application.

Instead, ask:

1. What did the archive try to preserve?
2. What media were being compared?
3. Why did public availability matter?
4. Why did the archive keep both English and Korean?
5. What does the structure reveal about its time?

If those questions still work, the archive still works.
