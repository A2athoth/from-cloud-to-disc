# RFC 0002: Public Does Not Mean Popular

## Summary

Something can be public without being widely seen.

Public means available.

Popular means noticed.

This repository is concerned with availability, not attention.

## Rationale

A small public record may matter even if almost no one reads it.

The value of an archive is not measured only by traffic, stars, or immediate usefulness.

Sometimes the important property is simpler:

it can be found.

## Principle

A public artifact does not need an audience to justify its existence.

It only needs a reason to be available.
