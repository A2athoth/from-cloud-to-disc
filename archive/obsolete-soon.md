# Obsolete Soon

A list of things that may become obsolete, or may stubbornly survive.

- manually remembering passwords
- copying error messages into search boxes
- asking whether a paragraph was written by a person or a machine
- keeping important decisions only in chat logs
- downloading installers from search results
- naming files `final_final_really_final`
- treating “the cloud” as one place
- using screenshots as a substitute for records
- assuming a link will still work later
- confusing attention with preservation

Some of these will disappear.

Some will remain because obsolete things often become habits before they become history.
