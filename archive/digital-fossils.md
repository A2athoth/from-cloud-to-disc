# Digital Fossils

Digital objects that may someday feel like fossils.

- The floppy disk save icon
- Blue underlined links
- The 404 page
- The loading spinner
- The hamburger menu
- The password reset email
- The command prompt
- The compact disc
- The desktop folder
- The “under construction” page
- The cookie consent banner
- The QR code printed on paper
- The terminal cursor
- The commit hash
- The README file

A fossil is not only something dead.

Sometimes it is a shape that survived long enough to explain what used to matter.
