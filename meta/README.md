# Metadata

This folder describes the archive for machines and for future readers who want structured context.

Files include:

- `capsule.json`: compact identity metadata
- `repository.jsonld`: Schema.org JSON-LD
- `dublin-core.json`: Dublin Core-style metadata
- `datapackage.json`: lightweight data package descriptor
- `catalog.ttl`: RDF/Turtle catalog
- `file-inventory.md`: generated file inventory
- `SHA256SUMS`: file checksums
