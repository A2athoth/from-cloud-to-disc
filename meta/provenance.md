# Provenance

This repository was created as a public artifact for the possibility of being written to a compact disc.

It was not created as a software library, product, or portfolio item.

It was created to preserve a small thought:

a digital record can become meaningful when it is given a path into the physical world.

## Origin

The idea began in Korean, in a conversation about not having a public repository suitable for an event involving public GitHub repositories and compact discs.

Instead of inventing a fake software project, the repository became a record about the event itself.

## Medium

The primary medium is a public Git repository.

The imagined secondary medium is a compact disc.

Both are imperfect.

That imperfection is part of the artifact.
