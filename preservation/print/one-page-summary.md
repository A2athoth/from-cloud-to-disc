# One-Page Summary

`from-cloud-to-disc` is a public plain-text time capsule.

It was made because a public repository could become a physical disc, and that movement from cloud to object felt worth marking.

It is not software.

It is a small archive about publicness, preservation, language, and media.

English is used for reach.
Korean is kept for origin.

The repository is complete when it can explain why it exists and how it wants to be read.
