# Accessibility

This page gathers the archive's low-friction reading notes: encoding, offline reading, no-JavaScript access, and plain-language summaries.

## Encoding

Files are intended to be read as UTF-8.

This matters because the archive contains Korean text.

If Korean text appears broken, check the text encoding before assuming the archive is damaged.

## No JavaScript Needed

This archive does not require JavaScript.

It does not require a build step.

It does not require a package manager.

A browser, text editor, terminal pager, or printed copy is enough.

## Offline Reading

The archive should remain understandable without network access after it has been copied.

Recommended offline entry points:

1. `README.md`
2. `README.ko.md`
3. `guide/COLLECTION.md`
4. `disc/readme-on-disc.md`
5. `preservation/accessibility/README.md`

External links are useful, but they should not be required for basic understanding.

## Plain-Language Summary

This is a small public archive made of text files.

It is about a digital record that may become a physical disc.

It does not need to be installed. It does not run code.

You can read it with a text editor or a web browser.

It uses English and Korean because the idea began in Korean, while the archive also wants to be readable by more people.
