# The Present

A few observations from this moment.

## Ordinary things

- People talk to machines and call it work.
- Code is still written mostly as text.
- Many important decisions disappear inside chat messages.
- The cloud is someone else's computer, but also where memory lives.
- Public does not always mean popular.
- Private does not always mean secret.
- We still lose files.
- We still forget passwords.
- We still rename things because naming is hard.
- We still trust logs more than memory.

## About software

Software is everywhere, but often invisible.

It wakes us up.  
It routes our messages.  
It stores our photographs.  
It counts our money.  
It recommends songs.  
It translates sentences.  
It breaks silently.  
It updates itself while we sleep.

Much of modern life is a conversation with systems we do not fully see.

## About records

Not everything needs to be permanent.

But when something is easy to delete, preserving even a small thing starts to feel intentional.

A text file is fragile.  
A repository is slightly less fragile.  
A disc is fragile in a different way.

This repository lives somewhere between those forms.

## About this repository

This repository was made because there was no public project to show, but there was still something to mark.

It is not here to impress.  
It is here to remain available.

That is enough.
