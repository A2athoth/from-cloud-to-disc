# README on Disc

This file is written for the possibility that the repository is being read from a physical disc.

## Suggested reading order

1. `README.md`
2. `MANIFEST.md`
3. `INDEX.md`
4. `notes-to-the-future.md`
5. `ko/README.md`
6. `meta/provenance.md`

## Verification

If the file `meta/SHA256SUMS` is present, it contains checksums for the files in this repository at the time it was generated.

Checksums do not make an archive meaningful.

They only help show whether the bits stayed still.
