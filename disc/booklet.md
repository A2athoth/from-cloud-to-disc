# Booklet

## From Cloud to Disc

This booklet is a plain-text substitute for something that might have been printed and folded into a CD case.

The repository is small on purpose.

It is meant to be opened without ceremony, copied without permission, and read without installing anything.

## Side A: The cloud

The cloud is where the record begins: distant, convenient, easy to change, easy to forget.

## Side B: The disc

The disc is where the record becomes awkwardly physical: limited, local, holdable, and still not permanent.

## The artifact

The artifact is not the disc alone.

It is the path between media.
