# Language

This page explains the language structure of the archive. English is used for reach; Korean is kept for origin.

## Origin Language

The idea for this archive began in Korean.

That matters because language is not only a container for meaning. It is also a record of where a thought first became speakable.

English lets the archive travel farther.

Korean keeps the archive closer to the moment of origin.

The archive needs both distances.

## Translation Notes

The Korean layer is not a strict mirror of the English layer.

Some sentences are translated closely.

Some are rewritten so they sound natural in Korean.

Some ideas appear only in Korean because they describe the origin of the archive more precisely there.

The goal is not perfect equivalence.

The goal is to preserve two readable surfaces of the same artifact.

## Korean Layer Map

The Korean layer appears in several forms:

- root-level Korean entrances: `README.ko.md`, `INDEX.ko.md`, `guide/COLLECTION.ko.md`
- Korean mirrors: `ko/present.md`, `ko/definitions.md`, and related files
- Korean-specific essays: `ko/essays/`
- Korean archive notes: `ko/archive/`
- language notes: `language/`

The Korean layer is not a single translation file.

It is a set of origins, mirrors, and native-language reflections.
