# 캡슐 메타데이터

이 문서는 저장소 자체를 설명합니다.

## 정체성

- 이름: `from-cloud-to-disc`
- 형식: 공개 GitHub 저장소
- 구성: 평문과 Markdown
- 의도한 매체: Git 저장소, 그리고 가능하다면 CD
- 목적: 소프트웨어 제품이 아닌 작은 공개 유물

## 원칙

어떤 것이 크거나 유용하거나 일반적인 의미에서 완성되어야만 보존할 가치가 생기는 것은 아닙니다.

이 저장소는 자신이 왜 존재하는지 설명할 수 있을 때 완성됩니다.

## 보존 메모

나중에 이 저장소가 발견된다면, 그것을 만들게 한 이벤트 자체는 더 이상 중요하지 않을 수도 있습니다.

중요한 것은 더 단순합니다.

누군가 디지털 기록이 물리적인 물건이 될 수도 있다는 사실을 알아차렸고, 그 순간을 표시하기로 했다는 것.
