# 한국어 레이어

이 폴더는 `from-cloud-to-disc` 저장소의 한국어 레이어입니다.

영어 문서의 단순한 직역이 아니라, 같은 생각을 한국어 감각으로 다시 남긴 기록입니다.

## 왜 분리했는가

루트 문서는 더 많은 사람이 읽을 수 있도록 영어 중심으로 두었습니다.

하지만 이 저장소의 출발점은 한국어였습니다.

그래서 한국어를 README의 작은 부록으로만 두지 않고, 별도의 레이어로 보존했습니다.
