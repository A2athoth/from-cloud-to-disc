# Guide

This folder contains navigation, review, and editorial guidance for the archive.

## Essential guides

- `REVIEWER_GUIDE.md` / `REVIEWER_GUIDE.ko.md`: two-minute review paths
- `READING_PATHS.md` / `READING_PATHS.ko.md`: purpose-based reading paths
- `ARCHIVE_MAP.md` / `ARCHIVE_MAP.ko.md`: conceptual map of the archive layers

## Supporting guides

- `BILINGUAL_BALANCE.md` / `BILINGUAL_BALANCE.ko.md`: language-balance policy
- `COLLECTION.md` / `COLLECTION.ko.md`: curated summary
- `FAQ.md` / `FAQ.ko.md`: quick questions

## Internal review notes

- `ROOT_STRUCTURE_REVIEW.md`: root layout decision
- `REVIEW_TEST.md` / `REVIEW_TEST.ko.md`: two-minute review check
- `CORE_STATEMENTS.md`: controlled repeated statements

General readers can start with `REVIEWER_GUIDE.md` and skip the internal review notes.

## Curation

- `curation/`: growth and selection rules
