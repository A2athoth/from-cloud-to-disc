# Core Statements

These are the few statements the archive should repeat, and only when useful.

1. A repository does not need to contain code to contain meaning.
2. English is used for reach; Korean is kept for origin.
3. The artifact is the movement from cloud to disc, not the disc alone.
4. Public means available, not necessarily popular.
5. The best version is the one that can still be understood later.

When a new file repeats these ideas, it should add context rather than decoration.
