# From Cloud to Disc

공개 GitHub 저장소가 클라우드의 기록에서 물리적인 디스크로 이동할 수 있다는 가능성을 기록한 이중언어 평문 타임캡슐입니다.

소프트웨어 제품도, 라이브러리도, 포트폴리오도 아닙니다. 매체, 보존, 언어, 발견가능성에 관한 작은 공개 아카이브입니다.

## 여기서 시작하기

- **영어 검토:** [`REVIEWER_GUIDE.md`](./REVIEWER_GUIDE.md)

- **제출 요약:** [`submission/short-description.ko.md`](./submission/short-description.ko.md)

## 왜 만들었나

보여줄 만한 일반적인 공개 프로젝트가 없었습니다. 그래서 억지로 프로젝트를 꾸미는 대신, 그 상황 자체를 기록했습니다.

저장소는 코드만 담는 장소가 아닐 수 있습니다. 어떤 순간의 타임스탬프가 될 수도 있고, 메시지가 될 수도 있고, 아카이브가 될 수도 있습니다. 그리고 클라우드 밖으로 나와 물리적인 매체가 될 가능성을 얻을 때, 조금 다른 의미를 갖게 됩니다.

## 언어에 대하여

영어는 더 멀리 읽히기 위해 사용했습니다.  
한국어는 이 생각의 출발점을 보존하기 위해 남겼습니다.  
둘 다 이 기록의 일부입니다.
