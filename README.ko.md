# From Cloud to Disc

이 저장소는 작은 공개 타임캡슐입니다.

클라우드에 있던 디지털 기록이 언젠가 CD라는 물리적인 물건이 될 수도 있다는 가능성 때문에 만들어졌습니다.

완성된 제품도 아니고, 라이브러리도 아니고, 포트폴리오용 프로젝트도 아닙니다.

그저 공개된 공간에 남겨둔 작은 기록입니다.

## 이 저장소의 생각

저장소는 코드만 담는 장소가 아닐 수 있습니다.

어떤 순간의 타임스탬프가 될 수도 있고,  
미래의 누군가에게 보내는 메시지가 될 수도 있고,  
작은 아카이브가 될 수도 있습니다.

## 언어에 대하여

이 저장소는 영어를 중심으로 구성되어 있습니다. 더 많은 사람이 읽을 수 있게 하기 위해서입니다.

하지만 이 아이디어는 한국어에서 출발했습니다. 그래서 한국어판과 한국어에 대한 기록도 별도로 남겼습니다.

영어는 도달 범위를 위해 사용했습니다.  
한국어는 출처를 위해 남겼습니다.  
둘 다 이 기록의 일부입니다.
