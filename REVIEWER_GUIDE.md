# Reviewer Guide

A two-minute guide to this repository.

## What this is

`from-cloud-to-disc` is a bilingual plain-text time capsule about a public GitHub repository becoming meaningful when it can move from cloud storage toward a physical disc.

It is not software. It is a small public archive built in the place where software usually lives.

## Why it exists

There was no conventional public project ready to publish.

Instead of inventing one, the repository records that absence directly and turns it into an artifact about publicness, preservation, language, and media.

## Why a disc matters

A disc is interesting here because it is physical, limited, offline, and no longer the default.

If a cloud-hosted repository becomes a disc, the record gains a different kind of presence.

## Why bilingual

English lets the archive travel farther.

Korean preserves the language in which the idea began.

## What makes it complete

It explains what it is, why it exists, how to read it, how to preserve it, and how it changed.
